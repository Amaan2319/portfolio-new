This is a full-stack blogging platform built with the MERN stack. Users can:

  • Register and log in with JWT-based authentication
  • Create, edit, and delete blog posts (CRUD operations)
  • Responsive UI built with React
  • Secure API routes using Express.js & MongoDB

Admin Dashboard

The application also includes an Admin Dashboard where admins can:

  • View all registered users
  • Manage blog posts (approve, delete, or edit)
  • Track platform activity in real time
